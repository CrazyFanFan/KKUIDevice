# KKUIDevice

["Hardware string" source](https://en.wikipedia.org/wiki/List_of_iOS_devices#Apple_Watch)


# Usage


## swift

### cocoapods

    target 'KKUIDevice-swiftDemo' do
      use_frameworks!
      pod 'KKUIDevice', :path => './'
    end

### eg.

    NSLog("%@", UIDevice.current.kk.hardware)
    NSLog("%@", UIDevice.current.kk.model)
